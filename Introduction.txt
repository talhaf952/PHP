PHP is a server scripting language and a powerful tool for making daynamic and interactive web pages.
Server-side scripting languages are a type of programming language that runs on the server instead of the client.

<?php
echo "Hello World!";
?>

What is PHP?
PHP is an acronym for "PHP: Hypertext Preprocessor"
PHP is a widely-used, open source scripting language
PHP scripts are executed on the server
PHP is free to download and use
